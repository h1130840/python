# 🍅 個人化番茄鐘與任務看板

這是我使用 Python 製作的專題報告，結合「時間管理」與「待辦清單」的實用網頁工具。

## 功能亮點 

* **任務管理**：
  * **新增**：可輸入並新增每日的待辦任務。
  * **讀取**：清楚列出所有任務狀態。
  * **更新**：打勾即可將任務標記為完成 (加上刪除線)。
  * **刪除**：一鍵刪除不需要的任務。
* **番茄鐘計時器**：內建專注倒數計時器 (預設 25 分鐘)，時間到會在網頁上觸發氣球動畫提醒。
* **資料持久化 (File I/O)**：任務進度會自動儲存在背景的 `tasks.json` 檔案中。

## 使用技術與套件

* **程式語言**：Python 3
* **網頁框架**：[Streamlit](https://streamlit.io/) (負責將 Python 邏輯轉化為前端 UI)
* **內建模組**：`json` (資料存取)、`os` (檔案路徑判斷)、`time` (計時器運算)
